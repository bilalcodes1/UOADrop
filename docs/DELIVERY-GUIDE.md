# دليل تسليم UOADrop لسعد

## الملفات المطلوبة

| الملف | الموقع | الاستخدام |
|-------|--------|-----------|
| `UOADrop Setup 0.1.0.exe` | `apps/desktop/release/` | تثبيت على Windows |
| `UOADrop 0.1.0.exe` | `apps/desktop/release/` | نسخة محمولة (بدون تثبيت) |

## خطوات التثبيت على جهاز سعد (Windows)

1. **انسخ** `UOADrop Setup 0.1.0.exe` على فلاشة USB
2. **شغّل** الملف على جهاز سعد
3. إذا طلع تحذير Windows SmartScreen: اضغط **More info** ← **Run anyway**
4. اختر مسار التثبيت (الافتراضي مقبول) ← **Install**
5. **شغّل التطبيق** من قائمة Start أو سطح المكتب

## أول تشغيل

- **PIN الدخول**: `12345`
- التطبيق يبدأ بشاشة القفل — ادخل الـ PIN
- الداشبورد يظهر فارغ (لا توجد طلبات بعد)

## اختبار الرفع المحلي (من داخل المكتبة)

1. من جهاز سعد — افتح المتصفح وادخل `http://localhost:3737`
2. يظهر **ملصق الحائط** بـ QR code
3. من **موبايل أي طالب** على نفس شبكة Wi-Fi:
   - امسح الـ QR أو ادخل العنوان المعروض (مثل `http://192.168.1.X:3737`)
   - ارفع ملف PDF أو صورة
   - اختر إعدادات الطباعة
   - اضغط **إرسال**
4. **بجهاز سعد**: يظهر إشعار + الطلب يظهر بالداشبورد
5. اضغط **طباعة** ← يفتح الملف بالتطبيق الافتراضي

## اختبار الرفع الأونلاين (من خارج المكتبة)

1. من أي جهاز ادخل: **https://uoadrop.vercel.app**
2. ارفع ملف + اكتب الاسم + البريد
3. بجهاز سعد: الطلب يظهر بعد ثوانٍ بعلامة 🌐

## اختبار الإشعارات

- **Telegram**: بعد رفع طلب أونلاين — يوصل تنبيه على البوت
- **Email**: يوصل إيميل عند تغيير حالة الطلب

## استكشاف المشاكل

| المشكلة | الحل |
|---------|------|
| التطبيق ما يفتح | Right-click ← Run as Administrator |
| SmartScreen يمنع | More info ← Run anyway |
| الطلبات الأونلاين ما تظهر | تأكد من اتصال الإنترنت |
| الموبايل ما يقدر يوصل | تأكد إن الموبايل وجهاز سعد على نفس شبكة Wi-Fi |
| الطابعة ما تطبع | تأكد إن الطابعة معرّفة كـ Default Printer بالنظام |
